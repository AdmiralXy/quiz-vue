## Quiz Vue

### About

This project implements a quiz using Vue.js and Bootstrap 5.

### Requirements & usage

Project requires: -

Tested on:
* Windows 10

Usage:
* Just open index.html in any internet browser like Google Chrome.

    ├── index.html
